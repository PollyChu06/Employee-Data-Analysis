# Employee Data Analysis Project
This repository contains my first Data Analysis project using Excel, named "Employee-Data-Analysis." The project demonstrates essential steps in the data analysis process, including Data Cleaning and Data Visualization.
[Click Here](https://github.com/PollyChu06/Employee-Data-Analysis/blob/main/Employee%20Data%20Analysis.xlsx) for the Project.

## Basic Steps in Data Analysis Project on Excel:

### Data Cleaning:

Remove Duplicates: Ensure the dataset does not contain duplicate entries.

Handle Missing Values: Identify and fill missing data appropriately.

Correct Inconsistencies: Standardize data formats and correct any errors.

### Data Transformation:

Format Conversion: Convert raw data into a more suitable format for analysis.

Create New Columns: Derive new columns based on existing data.

Aggregate Data: Summarize data to identify overall trends.

### Data Analysis:

Statistical Functions: Apply functions like SUM, AVERAGE, and COUNT to derive insights.

Pivot Tables: Use pivot tables to summarize and explore data from different perspectives.

### Data Visualization:

Charts and Graphs: Create bar charts, line graphs, and pie charts to visualize data.

Conditional Formatting: Highlight key data points and trends.

Dashboards: Develop comprehensive dashboards to present findings in an interactive format.

## Project Highlights:

Employee Demographics: Analyze the age, gender, and department distribution of employees.

Performance Analysis: Assess employee performance based on various metrics.

Salary Trends: Examine salary distribution and identify patterns over time.
